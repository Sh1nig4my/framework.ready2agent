# R2A Method Documentation

## Metadata

- Ultimo aggiornamento: 2026-04-28
- Pubblico: coding agent, maintainer, reviewer
- Status: attivo

## Scopo

`documentation/method/` descrive come Ready2Agent evolve con un metodo agent-native tracciabile.
Qui vivono regole operative, modelli di allineamento, onboarding e maintenance.

## Struttura

- `agent-onboarding/` -> onboarding operativo agenti
- `execution-rules/` -> regole esecutive e ordering
- `workflow-model/` -> relazione tra metodo e istanze workflow
- `prompt-model/` -> relazione tra metodo e prompt operativi
- `alignment-model/` -> stato/gap/rischi e priorita di riallineamento
- `maintenance-model/` -> principi di adozione e governance agent-assisted

## Relazione con workflow e prompts

- `documentation/method/` contiene teoria, regole e modelli.
- `workflow/` contiene le istanze operative concrete del metodo.
- `prompts/` espone l'interfaccia operativa con cui gli agenti avviano i task.
