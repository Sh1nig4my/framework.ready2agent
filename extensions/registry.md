# Extensions Registry

Registro ufficiale delle extensions disponibili (dev-side, non-runtime).

Stato iniziale: nessuna extension registrata.

## Regole

- ogni extension deve avere una entry in questo file;
- usare `extension-id` in kebab-case;
- dichiarare sempre stato e owner tecnico;
- il registro non abilita runtime loading.
