# Extension Template

Template ufficiale per una extension R2A dev-side (non-runtime).

Compilare:

- `extension.config.ts`
- `schema.ts`
- `permissions.ts`
- `docs.md`

Regole:

- naming cartella extension in kebab-case;
- nessuna dipendenza runtime dal Core;
- aggiornare sempre `extensions/registry.md`.
- non inserire esempi dominio-specifici nel template.
